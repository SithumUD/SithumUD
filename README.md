# 💫 About Me:
Hey there! 👋 I'm **Sithum Udayanga**, a passionate **Full Stack & Cloud Solutions Engineer** from Sri Lanka. I specialize in designing, developing, and deploying scalable software solutions for modern businesses, with a strong focus on backend systems, cloud architecture, and enterprise application development.

💻 I build and deliver end-to-end applications across **web, mobile, and desktop platforms**, transforming ideas into production-ready systems. My expertise includes **Java, Spring Boot, React.js, Next.js, Node.js, C# .NET Core, and RESTful API development**, with a strong focus on performance, security, and scalability.

☁️ I have hands-on experience working with **cloud platforms and DevOps tools**, deploying applications using **Docker** on platforms such as **Google Cloud Platform (GCP), Oracle Cloud Infrastructure (OCI), and DigitalOcean**. I also implement secure authentication systems using **JWT, role-based access control, and Keycloak**.

🏗️ I design and develop **scalable enterprise systems** such as Hotel Management Systems, POS Systems, Property Management Platforms, Booking Systems, and custom business applications tailored to real-world needs.

🚀 I continuously improve my skills in **software architecture, microservices, cloud engineering, and full-stack development**, while exploring modern technologies that solve real business challenges.

📩 **Let's connect!** I'm open to freelance projects, collaborations, and software development opportunities.
